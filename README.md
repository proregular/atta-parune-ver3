## 프로젝트 소개
>직장인들의 한정된 점심시간을 보다 효율적으로 활용할 수 있도록, 사전 예약 기능을 제공하는 식당 예약 솔루션을 개발했습니다. 사용자는 원하는 식당을 미리 예약하여 대기 시간을 줄일 수 있으며, 평균 식사시간이 짧은 식당을 빠르게 찾을 수 있도록 정렬 기능도 지원합니다.
